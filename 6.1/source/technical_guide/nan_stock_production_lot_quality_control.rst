.. i18n: Description
.. i18n: -----------
..

Description
-----------

.. i18n: ::
.. i18n: 
.. i18n:   This module adds two quality controls to production lots on arrival and after production. The idea is that the first
.. i18n:   test will be 'Generic' and check for things like correct packaging. The second will be specific to the product in question.
..

::

  This module adds two quality controls to production lots on arrival and after production. The idea is that the first
  test will be 'Generic' and check for things like correct packaging. The second will be specific to the product in question.
