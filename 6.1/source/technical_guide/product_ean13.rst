.. i18n: Description
.. i18n: -----------
..

Description
-----------

.. i18n: ::
.. i18n: 
.. i18n:   This module:
.. i18n:     * Adds a wizard on product form to generate valid EAN13 product codes.
.. i18n:     * By default EAN13 will be generated with Spanish country code (84) and 11111 company code. 
.. i18n:   Each company should change these values on administration/sequences to their country and company code.
.. i18n:   
.. i18n:   Este módulo:
.. i18n:     * Añade un asistente al producto para crear un código EAN13 con formato válido.
.. i18n:     * Por defecto, el código EAN13 será generado con la secuencia de país 84 y el código de empresa 1111.
.. i18n:   Cada compañía deberá adecuar estos valores por defecto a los de su país y empresa en Administración/Personalización/Secuencias.
.. i18n:   La secuencia de productos por defecto se inicia en 1 (y relleno de 5 dígitos tal como recomienda AECOC)
..

::

  This module:
    * Adds a wizard on product form to generate valid EAN13 product codes.
    * By default EAN13 will be generated with Spanish country code (84) and 11111 company code. 
  Each company should change these values on administration/sequences to their country and company code.
  
  Este módulo:
    * Añade un asistente al producto para crear un código EAN13 con formato válido.
    * Por defecto, el código EAN13 será generado con la secuencia de país 84 y el código de empresa 1111.
  Cada compañía deberá adecuar estos valores por defecto a los de su país y empresa en Administración/Personalización/Secuencias.
  La secuencia de productos por defecto se inicia en 1 (y relleno de 5 dígitos tal como recomienda AECOC)
