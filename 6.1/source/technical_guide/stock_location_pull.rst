Description

.. i18n:  Manages product's path in locations.  This module may be useful for different purposes: * Manages the product in his whole manufacturing chain * Manages different default locations by products * Define paths within the warehouse to route products based on operations:    - Quality Control    - After Sales Services    - Supplier Return * Manage products to be rent.
