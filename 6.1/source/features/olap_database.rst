Olap Database
=============

.. i18n: OpenERP's Olap module supports a fully-featured hybrid OLAP database, which
.. i18n: can connect to any relational database to perform multi-dimensional analysis.
.. i18n: MDX queries are supported by the web-services provided
.. i18n: by Open Object.
..

OpenERP's Olap module supports a fully-featured hybrid OLAP database, which
can connect to any relational database to perform multi-dimensional analysis.
MDX queries are supported by the web-services provided
by Open Object.
