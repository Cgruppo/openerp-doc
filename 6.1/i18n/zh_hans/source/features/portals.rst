Portals
=======

.. i18n: The portal module allows you to provide system access to customers
.. i18n: or suppliers. Show only a part of your OpenERP application without exposing
.. i18n: all modules of your system. It is a useful module to allow customers, partners,
.. i18n: suppliers, ... to use your system for particular tasks. 
..

The portal module allows you to provide system access to customers
or suppliers. Show only a part of your OpenERP application without exposing
all modules of your system. It is a useful module to allow customers, partners,
suppliers, ... to use your system for particular tasks.
