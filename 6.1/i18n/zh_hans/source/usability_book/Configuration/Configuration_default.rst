=====================
配置默认值
=====================

.. i18n: To Config the user's default, Click :menuselection:`Settings --> Technical --> Actions --> User-defined Defaults`.
..

要配置用户默认值, 使用菜单项: :menuselection:`配置(Settings) --> 技术(Technical) --> 动作(Actions) --> 用户定义默认值(User-defined Defaults)`.
