Description
-----------

::

  Module for Hotel/Resort/Restaurant management. You can manage:
      * Configure Property
      * Restaurant Configuration
      * table reservation
      * Generate and process Kitchen Order ticket,
      * Payment
  
      Different reports are also provided, mainly for Restaurant.
