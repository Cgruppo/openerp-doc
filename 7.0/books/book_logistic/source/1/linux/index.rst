
#############################
OpenERP Installation on Linux
#############################

	The installation procedure of OpenERP V6.0 is explained in this chapter. This procedure is well tested on Ubuntu
	version 10.04.

	There are four main procedures that you have to follow in order to run OpenERP V6.0 smoothly:

	* PostgreSQL Server Installation and Configuration

	* OpenERP Server Installation

	* OpenERP Client Installation

	* OpenERP Web Installation
